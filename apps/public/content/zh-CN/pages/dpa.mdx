_最后更新：2026年5月21日_

本数据处理协议（“DPA”）并入并构成 OpenPanel AB（“OpenPanel”、“我们”）与客户（“控制者”、“您”）之间 OpenPanel 服务条款的一部分。该协议适用于 OpenPanel Cloud 服务在代表您处理个人数据的情形。

## 1. 定义

- **GDPR** 指欧洲议会和理事会第 2016/679 号法规（EU）。
- **控制者** 指您，即决定处理目的和方式的客户。
- **处理者** 指代表您处理数据的 OpenPanel。
- **个人数据**、**处理**、**数据主体**和**监督机构**具有 GDPR 中赋予的含义。
- **分处理者** 指 OpenPanel 为提供服务而聘用的任何第三方，用于处理个人数据。

## 2. 我们的隐私处理方式

OpenPanel 的设计目标是尽可能减少个人数据收集。我们不使用 cookie 进行分析跟踪。我们不存储 IP 地址。相反，我们会使用访客 IP 地址、用户代理和项目 ID 的单向哈希，并结合每 24 小时轮换一次的盐值，生成一个每日轮换的匿名标识符。原始 IP 地址会立即丢弃，在盐值轮换后，该标识符将无法反向还原。

我们按事件存储的数据包括：

- 页面 URL 和来源页
- 浏览器名称和版本
- 操作系统名称和版本
- 设备类型、品牌和型号
- 城市、国家和地区（在请求时根据 IP 推导；随后 IP 会被丢弃）
- 您选择发送的自定义事件属性

没有持久标识符，没有 cookie，没有跨站跟踪。

基于这种方式，OpenPanel 在标准网站跟踪模式下收集的分析数据不构成 GDPR 第 4(1) 条所定义的个人数据。然而，我们提供此 DPA，供需要将其用于自身合规文档和处理活动记录的客户使用。

**会话回放（可选功能）**

OpenPanel 可选支持会话回放，且必须由控制者明确启用。启用后，会话回放会使用 rrweb 记录控制者网站上的 DOM 快照和用户交互（鼠标移动、点击、滚动）。这些数据会与会话标识符关联存储，并可能偶然捕获页面中可见的个人数据（例如，UI 中显示的已登录用户姓名）。所有文本内容和表单输入默认都会被遮蔽。控制者有责任确保其使用会话回放符合适用隐私法律，包括向终端用户提供适当通知。可通过 SDK 配置提供额外的遮蔽选项。

**AI 功能（可选，需主动启用）**

OpenPanel 提供可选的 AI 功能（例如对控制者分析数据的自然语言查询、异常洞察以及 AI 辅助报告），必须由控制者明确调用。当控制者使用这些功能时，提示词以及控制者分析数据的相关片段会被传输至 OpenAI, L.L.C.（美国）作为分处理者，仅用于生成所请求的响应。OpenAI 在其数据处理附录和欧盟标准合同条款下负有合同义务；OpenAI 不会使用通过 API 提交的数据训练其模型。如果控制者不使用这些功能，则不会向 OpenAI 发送任何数据。

## 3. 范围和角色

当 OpenPanel 代表控制者处理数据时，OpenPanel 作为**处理者**。对于从您网站和应用程序访客处收集的分析数据，您作为**控制者**。

## 4. 处理者义务

OpenPanel 承诺如下：

- 仅按照您的书面指示处理个人数据，不得用于任何其他目的。
- 确保所有可访问个人数据的人员均受适当的保密义务约束。
- 按照本 DPA 第 7 节实施并维持技术和组织措施。
- 未经您事先的通用或特定书面授权，不得聘用分处理者，并向任何分处理者传递同等的数据保护义务。
- 在合理可行的范围内，协助您回应数据主体行使其 GDPR 权利的请求。
- 在知悉个人数据泄露后，毫不无故延迟地通知您，且最迟不超过 48 小时。
- 提供所有必要信息，以证明符合本 DPA。您或您指定的审计员每 12 个月最多可审计一次（除非监管机构要求或在确认发生个人数据泄露后），且需提前 30 天通知，在保密条件下进行，由您承担费用，并且不得对运营造成不合理干扰。如现场审计无法提供，则可接受具有同等保证的书面文件请求替代。
- 在您选择下，于服务终止时删除或返还所有个人数据。

## 5. 您作为控制者的义务

您确认：

- 您对本 DPA 所述处理具有合法依据。
- 您已向终端用户提供适当的隐私声明。
- 您对指示 OpenPanel 处理的数据的准确性和合法性负责。

## 6. 分处理者

OpenPanel 使用以下分处理者来提供服务。所有分处理者要么位于欧洲经济区内，要么根据 GDPR 第 V 章（标准合同条款）提供充分保障。

| 分处理者 | 目的 | 位置 |
|---|---|---|
| Hetzner Online GmbH | 云基础设施和主要数据存储 | 德国（欧盟） |
| Cloudflare, Inc. | 公共端点前的 CDN、WAF 和边缘代理 | 欧盟边缘（任何经美国中转的数据适用 SCCs） |
| Cloudflare R2 | 备份存储 | 欧盟 |
| OpenAI, L.L.C. | LLM 提供商，仅在控制者调用可选 AI 功能时使用 | 美国（适用 SCCs） |
| Resend, Inc. | 事务性电子邮件投递（账户、账单和产品通知） | 美国（适用 SCCs） |

我们将提前合理通知您对此列表的任何拟议变更（新增或替换），并给予您提出异议的机会。

## 7. 技术和组织措施

OpenPanel 根据 GDPR 第 32 条实施以下措施：

**数据最小化和匿名化**
- IP 地址绝不存储。它们仅用于推导地理位置并生成匿名的每日标识符，随后即被丢弃。
- 每日轮换的加密盐值确保访客标识符在 24 小时后无法被还原或关联到个人。
- 不使用 cookie 或持久性的跨设备标识符。

**访问控制**
- 仪表板访问受身份验证和基于角色的访问控制保护。
- 生产系统仅授权人员可访问。

**加密和传输安全**
- 所有数据均通过 HTTPS（TLS）传输。

**基础设施和可用性**
- 所有数据托管于位于欧盟德国境内的 Hetzner 服务器上。
- 定期执行备份。
- 在正常运营过程中，数据不会离开欧洲经济区。

**事件响应**
- 我们维护用于检测、报告和调查个人数据泄露的流程。
- 如发生影响您数据的泄露事件，我们将在知悉后 48 小时内通知您。

**开源**
- OpenPanel 代码库公开托管在 GitHub 上，便于独立审查我们的数据处理实践。

## 8. 国际数据传输

所有分析数据均存储在位于德国的 Hetzner 基础设施上。备份存储在欧盟内的 Cloudflare R2 上。在标准运行情况下，分析数据不会离开欧洲经济区。

在以下情况下，可能会有有限的数据传输至美国：

- **事务性电子邮件**（通过 Resend, Inc.）——用于向控制者的授权用户发送账户、账单和产品通信所必需。
- **AI 功能**（通过 OpenAI, L.L.C.）——仅在控制者主动调用可选 AI 功能时发生，此时相关提示词和数据片段会被传输给 OpenAI 进行处理。
- **Cloudflare 边缘**——公共端点流量通过 Cloudflare 的全球网络提供服务。Cloudflare 的欧盟流量在可能的情况下会在欧盟边缘位置处理。

所有此类传输均受与各分处理者签署的欧盟标准合同条款（欧盟委员会决定 (EU) 2021/914）约束，提供 GDPR 第 V 章所要求的充分保障。

## 9. 数据保留和删除

**分析事件**会在您的账户处于活跃状态期间保留。我们目前不会对分析事件数据强制设定最长保留期限。如果未来引入保留限制，我们将提前通知所有客户。

**会话回放**保留 30 天，之后永久删除。

您可以随时在仪表板中删除单个项目、所有关联数据或整个账户。账户终止后，我们将在 30 天内删除您的数据，除非法律要求我们保留更长时间。

## 10. 管辖法律

本 DPA 受瑞典法律管辖，并依据 GDPR 进行解释。
