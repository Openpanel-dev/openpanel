只需插入以下代码片段，并将 `YOUR_CLIENT_ID` 替换为你的 client id。

```html title="index.html" /clientId: 'YOUR_CLIENT_ID'/
<script>
  window.op=window.op||function(){var n=[];return new Proxy(function(){arguments.length&&n.push([].slice.call(arguments))},{get:function(t,r){return"q"===r?n:function(){n.push([r].concat([].slice.call(arguments)))}} ,has:function(t,r){return"q"===r}}) }();
  window.op('init', {
    clientId: 'YOUR_CLIENT_ID',
    trackScreenViews: true,
    trackOutgoingLinks: true,
    trackAttributes: true,
  });
</script>
```

## 使用

### 跟踪事件

你可以通过两种不同的方法跟踪事件：直接调用 `window.op('track')`，或向 HTML 元素添加 `data-track` 属性。

```html title="index.html"
<button type="button" onclick="window.op('track', 'my_event', { foo: 'bar' })">
  跟踪事件
</button>
```

```html title="index.html"
<button type="button" data-track="my_event" data-foo="bar">跟踪事件</button>
```

### 识别用户

要识别用户，请使用唯一标识符调用 `window.op('identify')` 方法。

```js title="main.js"
window.op('identify', {
  profileId: '123', // Required
  firstName: 'Joe',
  lastName: 'Doe',
  email: 'joe@doe.com',
  properties: {
    tier: 'premium',
  },
});
```

### 设置全局属性

设置会随每个事件发送的属性：

```js title="main.js"
window.op('setGlobalProperties', {
  app_version: '1.0.2',
  environment: 'production',
});
```

### 增加属性值

为用户资料中的数值属性增加值。

- `value` 是要增加的数量。如果未提供，则默认为增加 1。

```js title="main.js"
window.op('increment', {
  profileId: '1',
  property: 'visits',
  value: 1 // optional
});
```

### 减少属性值

为用户资料中的数值属性减少值。

- `value` 是要减少的数量。如果未提供，则默认为减少 1。

```js title="main.js"
window.op('decrement', {
  profileId: '1',
  property: 'visits',
  value: 1 // optional
});
```

### 清除用户数据

清除当前用户的数据：

```js title="main.js"
window.op('clear');
```

## 高级用法

### 过滤事件

你可以通过向 `init` 方法添加 `filter` 属性来过滤事件。

下面是一个示例，用于禁用对在本地存储中包含 `disable_tracking` 项的用户的跟踪。

```js title="main.js"
window.op('init', {
  clientId: 'YOUR_CLIENT_ID',
  trackScreenViews: true,
  trackOutgoingLinks: true,
  trackAttributes: true,
  filter: () => localStorage.getItem('disable_tracking') === undefined,
});
```

### 使用 NPM 版 Web SDK

<Steps>
#### 第 1 步：安装 SDK

```npm
npm install @openpanel/web
```

#### 第 2 步：初始化 SDK

```js title="op.js"
import { OpenPanel } from '@openpanel/web';

const op = new OpenPanel({
  clientId: 'YOUR_CLIENT_ID',
  trackScreenViews: true,
  trackOutgoingLinks: true,
  trackAttributes: true,
});
```

#### 第 3 步：使用 SDK

```js title="main.js"
import { op } from './op.js';

op.track('my_event', { foo: 'bar' });
```
</Steps>

### Typescript

使用 SDK 时遇到 ts 错误？你可以在项目中添加一个自定义类型定义文件。

#### 简单方式

将以下代码粘贴到任意 `.d.ts` 文件中。

```ts title="op.d.ts"
declare global {
  interface Window {
    op: {
      q?: string[][];
      (...args: [
        'init' | 'track' | 'identify' | 'setGlobalProperties' | 'increment' | 'decrement' | 'clear',
        ...any[]
      ]): void;
    };
  }
}
```

#### 严格类型（来自 sdk）

创建一个 `op.d.ts` 文件并粘贴以下代码：
