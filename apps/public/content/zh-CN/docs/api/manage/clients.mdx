---
title: 客户端
description: 管理 OpenPanel 项目的 API 客户端。创建、读取、更新和删除具有不同访问级别的客户端。
---

## 身份验证

需要 `root` 客户端。请参阅 [身份验证](/zh-CN/docs/api/authentication) 指南。

## 基础 URL

```
https://api.openpanel.dev/manage/clients
```

## 客户端类型

| 类型 | 说明 |
|------|-------------|
| `write` | 接收事件和用户资料更新 |
| `read` | 查询分析数据（insights、export） |
| `root` | 完全访问权限 — 管理 API、读取和写入 |

## 端点

| 方法 | 路径 | 说明 |
|--------|------|-------------|
| `GET` | `/manage/clients` | 列出所有客户端（可选按 `projectId` 过滤） |
| `GET` | `/manage/clients/{id}` | 获取特定客户端 |
| `POST` | `/manage/clients` | 创建新客户端 |
| `PATCH` | `/manage/clients/{id}` | 更新客户端名称 |
| `DELETE` | `/manage/clients/{id}` | 永久删除客户端 |

客户端密钥仅在创建时返回一次，之后无法再检索。

完整的请求/响应 schema 请参阅 [API 参考](/zh-CN/docs/api-reference)。
