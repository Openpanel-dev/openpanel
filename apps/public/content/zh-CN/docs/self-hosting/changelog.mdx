## 1.0.0（稳定版）

OpenPanel 自托管现已进入稳定状态，未来不应再有破坏性变更。

如果你正在从旧版本升级，请留意日志，因为它会告诉你是否需要执行某些操作。虽然不是强制要求，但建议这样做，因为它之后可能会给你带来麻烦。

### 新环境变量

<Callout>
如果你正在从旧版本升级，并且想使用这些新变量，就需要编辑你的 `.env` 文件。
</Callout>

- `ALLOW_REGISTRATION` - 如果设置为 `false`，新用户将无法注册（只有第一个用户可以注册）。
- `ALLOW_INVITATION` - 如果设置为 `false`，新用户将无法被邀请。
- `RESEND_API_KEY` - 如果已设置，我们将使用 Resend 发送电子邮件。
- `EMAIL_SENDER` - 用于发送电子邮件的发件人地址。

### 移除了 Clickhouse Keeper

在 0.0.6 中，我们为 Clickhouse 引入了集群模式。这是一个错误，我们已将其移除。

从 services 和 volumes 中移除 op-zk

```
services:
  op-zk:
    image: clickhouse/clickhouse-server:24.3.2-alpine
    volumes:
      - op-zk-data:/var/lib/clickhouse
      - ./clickhouse/clickhouse-keeper-config.xml:/etc/clickhouse-server/config.xml
    command: [ 'clickhouse-keeper', '--config-file', '/etc/clickhouse-server/config.xml' ]
    restart: always
    ulimits:
      nofile:
        soft: 262144
        hard: 262144
volumes:
  op-zk-data:
    driver: local
```
