## ✨ 主要功能

- **🔍 進階分析**：[Funnels](/zh-TW/features/funnels)、cohort、使用者檔案與工作階段歷史
- **🎬 工作階段重播**：[錄製並重播使用者工作階段](/zh-TW/features/session-replay)，內建隱私控制
- **📊 即時儀表板**：即時資料更新與互動式圖表
- **🎯 A/B 測試**：內建變體測試與詳細拆分
- **🔔 智慧通知**：事件與 funnel 型警示
- **🌍 隱私優先**：無 Cookie 追蹤與 GDPR 合規
- **🚀 開發者友善**：完整 SDK 與 API 存取
- **📦 自架**：完整掌控你的資料與基礎架構
- **💸 透明定價**：沒有隱藏費用
- **🛠️ 自訂儀表板**：彈性的圖表建立與[資料視覺化](/zh-TW/features/data-visualization)
- **📱 多平台**：Web、行動裝置（iOS/Android）與伺服器端追蹤
- **🤖 MCP Server**：[讓 AI 了解你的使用者](/zh-TW/features/mcp) — 連接 Claude、Cursor 或任何 MCP 用戶端
- **💰 收入追蹤**：[監控購買與訂閱](/zh-TW/features/revenue-tracking)以及產品事件
- **🔌 整合**：[Google Search Console](/zh-TW/features/integrations)等更多功能

## 🔒 隱私優先

OpenPanel 的設計以隱私為核心：
- **不需要 Cookie** - 預設無 Cookie 追蹤
- **符合 GDPR 與 CCPA** - 為隱私法規而打造
- **支援自架** - 完整掌控你的資料
- **透明的資料處理** - 資料屬於你
